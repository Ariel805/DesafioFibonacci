Contexto del Problema

Muchas personas quieren ahorrar dinero, pero les cuesta mantener la constancia cuando el esfuerzo les parece demasiado grande desde el inicio. Si comienzas ahorrando una cantidad muy pequeña e incrementas el ahorro de forma gradual y natural, es más fácil mantener el hábito a largo plazo.

En este proyecto se propone un plan de ahorro donde cada mes se deposita la cantidad que corresponde a la Serie de Fibonacci. Esto significa que en los primeros meses el esfuerzo es mínimo (Bs. 1), pero con el tiempo los depósitos crecen de forma progresiva, siguiendo un patrón matemático que se encuentra en la naturaleza: el crecimiento de flores, conchas marinas, poblaciones de conejos y más.

Explicación del Algoritmo

La Serie de Fibonacci es una secuencia numérica donde cada número es la suma de los dos anteriores. Comienza con 1 y 1, y luego cada valor se obtiene sumando los dos anteriores:

1 + 1 = 2 → 3 → 5 → 8 → ...

En JavaScript, el algoritmo se construye con tres variables y un ciclo:

// Variables para guardar los dos valores anteriores
let a = 1;  // Mes 1
let b = 1;  // Mes 2
let c;      // Siguiente valor

for (let i = 3; i <= meses; i++) {
  c = a + b;  // Fibonacci: suma de los dos anteriores
  a = b;      // Avanzar una posición
  b = c;      // Avanzar una posición
}

Este algoritmo usa solo variables simples (sin arreglos), tal como se requiere en el desafío. Cada iteración del ciclo calcula el siguiente número de Fibonacci y actualiza las variables para la siguiente suma.

Conclusión

La Serie de Fibonacci demuestra cómo un patrón matemático puede transformarse en una herramienta práctica para la vida cotidiana. En este caso, aplicar Fibonacci al ahorro tiene dos ventajas clave:

🌱

Inicio suave

Los primeros meses se ahorra muy poco (Bs. 1), lo que hace el hábito fácil de comenzar sin presión económica.

📈

Crecimiento natural

El incremento sigue un patrón orgánico que existe en la naturaleza, por lo que no se siente arbitrario ni forzado.

🎯

Metas claras

El algoritmo permite conocer de antemano cuánto se acumulará al final del período planificado.

Este proyecto demuestra que las matemáticas no son solo teoría: son herramientas reales que nos ayudan a tomar mejores decisiones financieras y a entender los patrones del mundo que nos rodea.
