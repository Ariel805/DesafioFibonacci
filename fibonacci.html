Ahorro Progresivo

con la Serie de Fibonacci

Problema real - Crecimiento gradual de un hábito financiero

Ariel Orlando Bustillos Cadena

🌍

Contexto del Problema

Muchas personas quieren ahorrar dinero, pero abandonan el hábito cuando el esfuerzo inicial se siente demasiado grande. Una alternativa más realista es empezar con una cantidad pequeña y aumentar el monto poco a poco.

En este proyecto se propone un plan donde cada mes se deposita la cantidad que corresponde a la Serie de Fibonacci. Así, los primeros depósitos son suaves y luego crecen de manera progresiva: 1, 1, 2, 3, 5, 8...

Explicación del Algoritmo

La Serie de Fibonacci es una secuencia donde cada número se obtiene sumando los dos anteriores. Para calcular el ahorro de cada mes se guardan los dos últimos valores y se genera el siguiente con una suma.

1 + 1 = 2 → 3 → 5 → 8 
// Variables para guardar los dos valores anteriores
let a = 1;
let b = 1;
let c;

for (let i = 3; i <= meses; i++) {
  c = a + b;
  a = b;
  b = c;
}

El algoritmo permite saber cuánto se debe ahorrar cada mes y cuál será el total acumulado al terminar el plan.

Conclusión

Fibonacci ayuda a crear un plan de ahorro que empieza fácil y aumenta de manera natural. Este tipo de crecimiento demuestra que una regla matemática puede convertirse en una herramienta práctica para tomar mejores decisiones financieras.
