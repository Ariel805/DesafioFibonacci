Contexto del Problema

En sistemas de seguridad, simulación y análisis de datos a veces se buscan números con propiedades especiales. Un ejemplo interesante son los Fibonacci primos: números que aparecen en la Serie de Fibonacci y además son primos.

Son extremadamente raros. Por eso este explorador funciona como un buscador educativo de candidatos para códigos de respaldo o identificadores matemáticos: primero genera Fibonacci y luego filtra solo los que también pasan la prueba de número primo.

Explicación del Algoritmo

El algoritmo combina dos ideas: generar la Serie de Fibonacci y revisar si cada valor generado es primo. Solo se guardan como candidatos los números que cumplen ambas condiciones.

1 1 2 3 5 8 13 ... 
for (let i = 1; i <= terminos; i++) {
  valor = generarFibonacci(i);

  if (esPrimo(valor)) {
    guardarComoFibonacciPrimo(valor);
  }
}

Esta combinación permite observar una rareza matemática: no todos los Fibonacci son primos, y no todos los primos aparecen dentro de Fibonacci.

Conclusión

Los Fibonacci primos son un buen ejemplo de cómo dos conceptos matemáticos pueden combinarse para crear una búsqueda más interesante. En criptografía y seguridad se estudian patrones numéricos especiales, y este proyecto muestra de forma sencilla cómo filtrar una serie usando una propiedad de primalidad.
